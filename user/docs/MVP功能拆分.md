# 心晴（MindWeather）MVP 功能拆分

## MVP 概述

**MVP 目标**：用最少资源验证核心假设——"天气隐喻降低情绪表达门槛"。

**MVP 核心闭环**：注册 → 个人2D地图点击区域投稿 → AI识别情绪 → 天气映射 → 首页3D公共地图呈现 → 个人天气沉淀。

---

## MVP 范围（必须保留）

```
✅ AI情绪识别
✅ 情绪→天气映射
✅ 首页3D公共校园地图（上下两层：天气粒子效果 + 3D地图）
✅ 个人2D气象地图（点击区域投稿入口）
✅ 个人天气空间（今日天气 + 历史记录 + 简单统计）
```

## MVP 砍掉（不在本期）

```
❌ 好友系统
❌ 聊天
❌ 语音投稿 / 变声
❌ 治愈互动（小太阳/留言）
❌ 校园认证（保留简单邮箱注册）
❌ Redis（先直接查MySQL）
❌ 微信小程序（先做H5/Web）
❌ 情绪预测
❌ 复杂权限系统
❌ AI心理咨询
```

## P0 - MVP 核心功能

### 模块一：用户系统（极简版）

| 序号 | 功能 | User Story | 验收标准 |
|------|------|-----------|----------|
| US-01 | 注册 | 作为新用户，我用邮箱注册账号 | 输入邮箱→设置密码→注册成功（无需校园认证） |
| US-02 | 登录 | 作为注册用户，我用邮箱+密码登录 | 邮箱+密码→登录成功，获取Token |
| US-03 | 匿名昵称 | 系统自动生成匿名昵称 | 注册后随机分配「晴空旅人」「微风听者」等昵称 |

### 模块二：文字情绪投稿（通过个人2D地图入口）

| 序号 | 功能 | User Story | 验收标准 |
|------|------|-----------|----------|
| US-04 | 个人2D地图 | 作为用户，我在个人页看到2D校园地图 | 腾讯地图2D加载，显示6个区域标记 |
| US-05 | 点击区域投稿 | 作为用户，我点击地图区域即可投稿 | 点击区域→弹出输入框→输入文字→提交 |
| US-06 | 天气映射反馈 | 作为用户，提交后看到天气映射结果 | 提交→AI处理→展示「你的天气：小雨🌧」+ 情绪标签 |

### 模块三：AI情绪识别引擎

| 序号 | 功能 | User Story | 验收标准 |
|------|------|-----------|----------|
| US-07 | 文本情绪识别 | 系统输入文本后自动识别情绪 | DeepSeek API，返回情绪分类+置信度 |
| US-08 | 天气映射 | 系统根据情绪自动映射天气 | 开心→☀️/焦虑→☁/压力→🌧/低落→⛈ |
| US-09 | 情绪标签生成 | 系统从文本提取3-5个情绪标签 | 如「考试周」「ddl」「想家」等关键词 |
| US-10 | 区域天气聚合 | 定时聚合各区域投稿生成区域主天气 | 每30分钟统计各区域情绪分布 |

### 模块四：首页3D公共校园情绪地图

| 序号 | 功能 | User Story | 验收标准 |
|------|------|-----------|----------|
| US-11 | 3D场景加载 | 作为用户，打开首页看到3D校园模型 | Three.js加载，≤3秒，上下两层结构 |
| US-12 | 天气效果层 | 作为用户，看到上层天气粒子特效 | 云层飘动/下雨/晴天光照等，随区域天气实时变化 |
| US-13 | 下层3D地图 | 作为用户，看到下层3D校园建筑 | 6个区域建筑模型+天气图标标识 |
| US-14 | 3D地图交互 | 作为用户，可旋转、缩放3D地图 | 单指旋转、双指缩放 |
| US-15 | 区域详情 | 作为用户，点击区域查看情绪概况 | 弹窗：主天气图标+标签云+投稿人数 |

### 模块五：个人气象空间

| 序号 | 功能 | User Story | 验收标准 |
|------|------|-----------|----------|
| US-16 | 今日天气 | 作为用户，看到今日个人天气 | 大天气图标+一句话情绪描述 |
| US-17 | 投稿历史 | 作为用户，浏览历史投稿 | 按日期降序列表，每条含天气图标+标签 |
| US-18 | 简单统计 | 作为用户，看到本月天气分布 | 饼图：晴天/阴天/雨天占比 |

## MVP 上线判定标准

- [ ] 用户可以邮箱注册/登录
- [ ] 用户在个人2D地图上点击区域 → 输入文字 → 提交 → 看到天气映射结果
- [ ] 首页3D校园地图正常加载，天气效果层随区域数据实时变化
- [ ] 用户可旋转/缩放3D地图，点击区域查看情绪概况
- [ ] AI情绪识别准确率 ≥ 80%（内部评测）
- [ ] 单次投稿耗时 ≤ 30秒
- [ ] 无P0级别线上Bug
